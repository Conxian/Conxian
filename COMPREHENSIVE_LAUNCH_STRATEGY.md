**Epoch Phases:**

1. **Accumulation Phase** (Days 1-28): Points earned

### 4.2 Epoch Transitions

**Automated Tasks:**

```clarity

  (begin
    ;; 1. Finalize previous epoch
    (try! (contract-call? .points-oracle finalize-epoch (- epoch-id u1)))
    
    ;; 2. Calculate conversion rates
    (try! (calculate-conversion-rates epoch-id))
    
    ;; 3. Open claim window
    (try! (contract-call? .gamification-manager open-claim-window epoch-id))

    (ok true)
  )
)
```

- Founder reviews epoch calculations (Epochs 1-4)

- Emergency pause available if issues detected

### 4.3 Points Calculation Process

**Off-Chain Service:**

```typescript
// points-calculator.ts
async function calculateEpochPoints(epochId: number) {
  // 1. Fetch all on-chain events
  const events = await fetchEpochEvents(epochId);
  
  // 2. Calculate points per user
  const userPoints = calculateUserPoints(events);
  
  // 3. Apply behavior multipliers
  const adjustedPoints = applyMultipliers(userPoints);
  
  // 4. Generate Merkle tree

  // 5. Submit root (3-of-5 multisig)

  return merkleTree;
}
```
