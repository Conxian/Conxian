# Conxian PRD (Spec-Driven Design)

This folder contains product requirements (PRD) and spec-driven design documents aligned with the current codebase.

- Centralized traits: `contracts/traits/all-traits.clar`
- Conventions: `use-trait alias .all-traits.<trait>` + `impl-trait .alias`
- This PRD reflects implemented contracts and planned items, clearly labeled.

## Index

- TRAITS.md — Centralized trait catalog and usage conventions
- TOKENS.md — CXD, CXVG, CXLP, CXTR, CXS
- VAULT.md — Vaults and strategies
- DEX.md — Factory, pools, router
- GOVERNANCE.md — Access control and governance
- ORACLE.md — Price oracles and aggregators
- SECURITY.md — Circuit breaker, pausable, monitoring
- MEV.md — Commit-reveal and batch ordering
